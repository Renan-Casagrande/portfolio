EchoBubble

Em Desenvolvimento Em breve na Chrome Web Store

Extensão para navegadores baseados no Chrome que adiciona tradução visual em tempo real ao WhatsApp Web. O usuário continua vendo a mensagem original enquanto a tradução aparece logo abaixo em um balão próprio — sem copiar, sem colar, sem sair da conversa.

27 idiomas disponíveis. Mais idiomas em breve. Privado e seguro — seus dados ficam apenas no seu dispositivo.

JavaScript HTML CSS Chrome Manifest V3

Funcionalidades

Tradução automática das mensagens recebidas
Balões de tradução personalizados

Privado — dados ficam apenas no dispositivo
Funciona diretamente no WhatsApp Web

Configuração de idioma padrão
Diferentes estilos visuais para os balões
Mais idiomas em breve
